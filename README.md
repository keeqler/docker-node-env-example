# Requirements

- `docker`
- `docker-compose`

# Getting started

Clone this repository:

```shell
$ git clone http://github.com/keeqler/docker-node-env-example
```

Enter in the clone's directory

```shell
$ cd docker-node-env-example
```

# Running

Simply, run:

```shell
$ docker-compose up -d
```

Finally, access `localhost:3000` in your browser
